FULL WEBSITE SETUP (REBUILT UI + SAME FEATURES)

1) Open terminal in this folder:
   Bus Ticket booking System

2) First time only:
   npm install

3) Start server:
   npm start

4) Open in browser:
   Home page:
   http://localhost:3001/index.html

   DB Booking:
   http://localhost:3001/booking-db.html

   Ticket page:
   http://localhost:3001/ticket-view.html?id=1

   Admin table:
   http://localhost:3001/admin-bookings.html

Features kept same:
- Quick booking page (frontend only): booking.html
- DB booking save: booking-db.html
- Ticket details from DB: ticket-view.html
- All bookings table: admin-bookings.html

Database file:
- bookings.db

If port 3001 is busy:
- Run: $env:PORT=3002; npm start
- Then open URLs with 3002.
